## Typography

### Primary Typeface
**Inter** - A sleek geometric sans-serif
- Used for: Headlines, body text, UI elements
- Characteristics: Modern, highly legible, geometric precision
- Weights: Light (300), Regular (400), Medium (500), Bold (700)

### Secondary Typeface
**Spectral** - A serif accent for lore and ritual content
- Used for: District descriptions, philosophical content, special sections
- Characteristics: Elegant, readable, with character
- Weights: Regular (400), Medium (500), Bold (700)

### Type Scale
- H1: 64px / 4rem (Inter Bold)
- H2: 48px / 3rem (Inter Bold)
- H3: 32px / 2rem (Inter Medium)
- Lead: 24px / 1.5rem (Inter Regular)
- Body: 18px / 1.125rem (Inter Regular)
- Small: 14px / 0.875rem (Inter Regular)

### Primary Colors
- **Midnight Blue** (#0A0F1F) - Primary background, authority
- **Neon Cyan** (#00E5FF) - Interactive elements, highlights

### Secondary Colors
- **Slate** (#1C2333) - Cards, panels, secondary backgrounds

- **Lavender** (#C7B7FF) - Soft highlights, secondary accents
- **Teal Mist** (#A8FFF1) - Calm, healing, secondary soft accent

### Semantic Colors
- **Text Primary**: White (#FFFFFF) on dark backgrounds
- **Text Secondary**: Light Gray (#D8D8E8) for supporting text
- **Accent Primary**: Neon Cyan for interactive elements
- **Accent Ritual**: Warm Gold for special, meaningful moments

- **Glow Soft**: rgba(199,183,255,0.14) for ambient effects

## Accessibility

### Motion & Animation
- Respect `prefers-reduced-motion` settings
- Provide animation controls for users
- Use meaningful transitions, not decorative
- Test animations with vestibular disorder considerations

### Color & Contrast
- WCAG AA compliance (4.5:1 contrast ratio)
- Color-blind friendly palettes
- Multiple ways to convey information
- High contrast mode support

### Typography & Readability
- Minimum 18px body text on digital
- Generous line spacing (1.5x line height)
- Clear hierarchy with consistent sizing
- Dyslexia-friendly font choices
