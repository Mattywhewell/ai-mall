## ðŸŒŠ 1. Emotional Intelligence Engine

### What It Does

Detects user emotional state from behavioral signals and adapts the **entire experience** accordingly.

### How It Works

**Input Signals:**
- Browsing speed (slow = contemplative, fast = stressed/searching)
- Navigation pattern (scattered = overwhelmed, linear = purposeful)
- Search terms (language reveals emotional state)
- Time of day (3am browser = different needs than 2pm)
- Cart abandonment (indecision, uncertainty, fear)
- Chapel visits (which mystical spaces resonate?)

**Output:**
```typescript
{
  primary_emotion: 'seeking',
  intensity: 75,
  needs: ['clarity', 'inspiration', 'guidance'],
  recommended_journey: 'contemplative',
  chapel_affinity: 'Mystery',
  color_palette: ['#4a5568', '#667eea', '#764ba2'],
  music_tempo: 'slow'
}
```

### What Changes

Based on emotional state, the system adapts:

**Visual Layer:**
- Color palettes shift to match emotion
- Breathing patterns in animations change
- Spacing and density adjust (stressed = more whitespace)

**Content Layer:**
- AI Spirit greetings become empathetic
- Product descriptions emphasize different benefits
- Navigation suggestions change completely

**Product Scoring:**
- Emotional resonance added to scoring (0-100)
- Products that "feel right" get boosted
- Ritual suggestions included with every product

### Example

**User arrives at 2:47am, rapid clicking, searches "can't sleep", abandoned cart twice:**
